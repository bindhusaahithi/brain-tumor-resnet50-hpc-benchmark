# Upload Checklist

Before uploading to GitHub or attaching to a resume, verify:

- `README.md` is visible at the repository root.
- `images/` contains generated charts.
- `data/raw/` does not contain private or large MRI files.
- `artifacts/` does not contain model weights unless intentionally shared.
- `reports/metrics/` contains benchmark CSV files.
- `docs/model_card.md` and `SECURITY.md` are included.
- The resume links to the GitHub repository, not a local folder path.
- The repository description mentions ResNet50, brain MRI classification, CPU/GPU benchmarking, and Streamlit demo.
